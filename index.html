What the Mandelbrot Set is

The Mandelbrot Set is the collection of all complex numbers c for which the repeated process z0 = 0, zn+1 = zn2 + c does not run away to infinity.

z0 = 0,   zn+1 = zn2 + c

Each pixel in the image corresponds to one value of c = a + bi on the complex plane. We test that pixel by starting at zero and applying the same rule again and again.

An adjacent pixel is tested by exactly the same procedure, but with a slightly different value of c determined by its neighbouring screen position. In the Mandelbrot Set the starting value is always z0 = 0; what changes from pixel to pixel is the parameter c, not the initial value of z.

  1. Choose a point c in the complex plane.
  2. Start with z = 0.
  3. Replace z by z2 + c.
  4. If |z| grows beyond 2, it will escape forever.
  5. If it stays bounded for many steps, the point is in or very near the Mandelbrot Set.

In practical computer graphics we only test a finite number of steps, so boundary points are approximated. That is why more iterations reveal finer detail.

Bounded vs unbounded

The black interior corresponds to points whose orbits remain bounded. The coloured bands outside mark points that escape, with colour determined by how quickly escape occurs.

Slow escape Faster escape
  • Inside: bounded or very likely bounded after many checks.
  • Outside: unbounded - the orbit eventually shoots away.
  • Boundary: extremely delicate; tiny changes in c can completely change the behaviour.

How the image is generated

The computation is repeated for every pixel. The resulting shape is governed by the extraordinarily intricate boundary between stable and unstable behaviour.

In this page the principal Mandelbrot image is not a stored picture. The browser on the local device carries out the iteration for each pixel and renders the image directly on the canvas. Consequently, the view shown on a PC, iPhone, or iPad is produced by computation on that device rather than by displaying a pre-generated fractal image.

An adjacent pixel is tested by exactly the same procedure, but with a slightly different value of c. The complex plane, often called the Argand plane, may be viewed as a coordinate plane: the horizontal axis gives the real part and the vertical axis gives the imaginary part. In that sense it is helpful to think of the image as a map. Adjacent pixels correspond to adjacent coordinates, so each neighbouring pixel represents a nearby complex number.

For the Mandelbrot Set the starting value is always z0 = 0. What changes from pixel to pixel is not the initial value of z, but the parameter c. If one pixel corresponds to c = -0.7500 + 0.1000i, then a neighbouring pixel might correspond to c = -0.7497 + 0.1000i or c = -0.7500 + 0.0997i, depending on whether the neighbouring pixel lies horizontally or vertically adjacent.

The test is then repeated independently for each such value of c: z1 = z02 + c, z2 = z12 + c, z3 = z22 + c, and so on. Because nearby values of c can have very different long-term behaviour, one pixel may escape quickly, the next may escape only after many iterations, and another may remain bounded. That sensitivity is exactly what produces the intricate boundary.

How each pixel is tested

The image is a rectangular window on the complex plane. A pixel at horizontal position px and vertical position py is converted into a complex number c = a + bi by mapping screen coordinates to the current view.

a = xcentre + (px/W - 1/2) · scale
b = ycentre + (py/H - 1/2) · scale · (H/W)

Once that pixel has been assigned its corresponding complex number c, the program performs exactly the same test for that pixel as for every other one:

  1. Set z0 = 0.
  2. Compute z1 = z02 + c = c.
  3. Compute z2 = z12 + c, then z3, and so on.
  4. If an iterate crosses the escape circle |z| = 2, the pixel is classified as outside the Mandelbrot Set.
  5. If no escape is detected after the chosen iteration limit, the pixel is coloured as interior or near-boundary.

The rendered image is therefore a map of long-term behaviour: each pixel records whether its associated orbit escapes, and if so how rapidly.

Why the radius-2 circle is used

The test uses the circle of radius 2 centred at the origin in the Argand plane. The relevant conditions are |c| > 2 or |z| > 2, meaning that the point lies outside that circle. Here the vertical bars denote absolute value, that is, distance from the origin in the complex plane.

Since z0 = 0, the first iterate is z1 = c. Therefore, if |c| > 2, the orbit has already crossed the radius-2 circle at the first step, so that value of c cannot belong to the Mandelbrot Set.

A second fact is fundamental for computation: once any later iterate satisfies |zn| > 2, the orbit will continue to grow and will not return to a bounded region. That is why the computation may stop as soon as the orbit crosses the circle.

|zn+1| = |zn2 + c| ≥ |zn|2 - |c|

For Mandelbrot testing one only needs to consider |c| ≤ 2, because larger values escape immediately. So after an iterate has moved outside the circle,

|zn+1| ≥ |zn|2 - 2

If |zn| > 2, then |zn|2 - 2 > |zn|. So the next term is even farther from the origin:

|zn+1| > |zn|

Once the orbit moves outside the radius-2 circle, the squaring term dominates and the sequence diverges to infinity rather than returning to a bounded region. The orbit diagram above shows this geometrically: after crossing the blue circle, successive points move outward instead of returning toward the origin.

Why the boundary supports unbounded detail

The boundary of the Mandelbrot Set is not a smooth curve. It contains structure at every scale, so increased magnification continues to reveal spirals, bulbs, filaments, and miniature Mandelbrot copies.

The structure is not exactly self-similar in the elementary sense of a simple repeating motif. Instead, the same iterative rule generates further layers of detail at progressively smaller scales.

This implementation permits substantial magnification within the browser, but any computation is ultimately limited by finite precision. The mathematical object is not limited in this way; only its numerical approximation is.

Why it matters in maths

  • Complex dynamics: it is a central object in studying repeated functions.
  • Julia sets: each value of c has a related Julia set, and the Mandelbrot Set organises which Julia sets are connected.
  • Chaos and stability: it shows how tiny changes in input can cause large qualitative changes in long-term behaviour.
  • Computation and visualisation: it is a canonical example of how algorithms render abstract mathematical structure visible.

This sensitivity is indeed closely related to the theory of nonlinear dynamical systems. The map z \mapsto z2 + c is nonlinear because of the squaring term. Nonlinear systems commonly exhibit a sharp distinction between stable and unstable behaviour, together with strong dependence on initial conditions or parameters. Near the boundary of the Mandelbrot Set, very small changes in the parameter c can alter the long-term behaviour of the orbit completely, which is why neighbouring pixels may display markedly different colours.

Three mathematical viewpoints

1. A set of numbers It is literally a set of complex numbers c satisfying a boundedness condition.
2. A stability map The picture marks which parameters lead to stable iteration and which lead to escape.
3. A geometric object The algebra defines a boundary with rich geometry and deep connections to other branches of mathematics.

Julia sets

For a fixed complex number c, the same iteration z \mapsto z2 + c can be studied for many different starting values of z, rather than always starting from 0. The boundary separating points whose orbits escape from those that remain bounded is the Julia set for that chosen value of c.

The Mandelbrot Set acts as a parameter map for this family: when c lies in the Mandelbrot Set, the corresponding Julia set is connected; when c lies outside it, the Julia set is disconnected and typically dust-like.

Benoit Mandelbrot

Benoit Mandelbrot (1924-2010) was a Polish-born French-American mathematician best known for his work on fractal geometry. He worked extensively at IBM and helped show that many irregular forms in mathematics, physics, finance, and nature can be studied through scaling and self-similar structure.

The set that now bears his name had been examined in earlier work by Gaston Julia and Pierre Fatou, but Mandelbrot’s use of computer graphics in the late twentieth century made its remarkable structure visible and widely understood. His work transformed fractals from a specialised topic into a major part of modern mathematical culture.
