Mandelbrot: pixel-by-pixel build

A fixed 100×100 logical-pixel view (same window as the main page’s starting view) is filled in a square spiral from the centre, 32 pixels per frame. Each step runs the usual escape test. If the orbit never has |z| > 2 within the iteration budget, there is no escape and this page colours the pixel black (very dark blue). Escaped pixels get bright colours from the escape speed. When the image is finished, move the pointer over the picture to inspect a pixel—handy next to a bright pixel touching a black one. Back to main Mandelbrot page
